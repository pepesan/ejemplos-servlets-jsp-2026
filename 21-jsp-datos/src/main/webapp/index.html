21 — Paso de datos Servlet → JSP

El patrón básico de MVC clásico: el servlet construye datos y los deposita en el request con setAttribute(); la JSP los lee con EL (\${clave}) y los presenta sin lógica de negocio.

Servlet: req.setAttribute("clave", valor)  →  forward(req, resp)  →  JSP: \${clave}
