19 — Subida de ficheros con @MultipartConfig

Demuestra cómo aceptar ficheros enviados con enctype="multipart/form-data" desde Servlet 3.0 usando la anotación @MultipartConfig y la API request.getPart(). No se necesita ninguna librería externa (Apache Commons FileUpload, etc.).

SubirFicheroServlet — subida con validación de tipo y tamaño

Formulario multipart/form-data con <input type="file"> y un campo de texto normal. El servlet valida el tipo MIME (imagen, PDF, TXT) y el tamaño (máx. 2 MB), extrae el nombre original del header Content-Disposition, sanea el nombre para evitar path traversal y guarda el fichero en el directorio temporal del servidor con Part.write().
