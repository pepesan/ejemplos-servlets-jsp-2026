62 – MVC Front Controller + Hibernate + H2

Mismo patrón que el módulo 32, añadiendo Hibernate como capa de persistencia:

Petición HTTP
  → FrontControllerServlet  (@WebServlet("/app/*"))
      → EmpleadoController.listar(req, resp)
           dao.listarTodos()   ← Hibernate consulta H2 en memoria
           req.setAttribute("empleados", lista)
           return "lista.jsp"
      → forward → WEB-INF/vistas/lista.jsp
                   ${empleados}  /  <c:forEach>  /  <fmt:formatNumber>

Acciones

  • Lista de empleados

Componentes clave

  • FrontControllerServlet — único servlet, tabla de rutas
  • EmpleadoController — lógica: listar, ver, guardar…
  • EmpleadoDao — acceso a datos con Hibernate Session
  • HibernateListener — inicia/cierra la SessionFactory
  • modelo/Empleado.hbm.xml — mapeo entidad ↔ tabla H2
