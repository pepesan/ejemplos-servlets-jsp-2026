32 – MVC: Front Controller + Controller + JSP/JSTL

Patrón MVC con Front Controller que delega en un controlador por recurso:

Petición HTTP
  → FrontControllerServlet  (único servlet, @WebServlet("/app/*"))
      → AlumnoController.listar(req, resp)
           req.setAttribute("alumnos",  lista)
           req.setAttribute("notaMedia", media)
           return "lista.jsp"
      → forward → WEB-INF/vistas/lista.jsp
                   ${alumnos}, ${notaMedia}, <c:forEach>…

Acciones

  • Lista de alumnos → AlumnoController.listar() → lista.jsp

Clases principales

  • FrontControllerServlet — mapea rutas a métodos del controller
  • AlumnoController — toda la lógica: listar, ver, guardar…
  • Accion — interfaz funcional: String ejecutar(req, resp)
  • modelo/Alumno + AlumnoRepositorio — datos en memoria
