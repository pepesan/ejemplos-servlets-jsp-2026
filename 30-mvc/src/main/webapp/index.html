30 — Patrón MVC con Front Controller

Patrón: Un único servlet (FrontControllerServlet) recibe todas las peticiones bajo /app/*, las enruta al comando correspondiente, y hace forward a la vista JSP.

Flujo de la petición

  Navegador
     │  GET /app/productos
     ▼
  FrontControllerServlet  (mapeo /app/*)
     │  pathInfo = "/productos" → ListarProductosComando
     ▼
  ListarProductosComando
     │  req.setAttribute("productos", ...)
     │  return "lista.jsp"
     ▼
  /WEB-INF/vistas/lista.jsp   (no accesible directamente)

Rutas disponibles

  GET  /app/productos          → lista de productos
  GET  /app/productos/ver?id=X → detalle de un producto
  GET  /app/productos/nuevo    → formulario de alta
  POST /app/productos/nuevo    → guardar y redirigir (PRG)
