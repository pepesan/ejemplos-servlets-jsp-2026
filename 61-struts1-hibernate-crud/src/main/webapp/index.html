61 — Struts 1.x + Hibernate: CRUD completo

⚠️ Requiere JDK 8. Struts 1.3.x y Hibernate 3.6.x son incompatibles con Java 9+.
Resumen del curso: Struts 1.x (DispatchAction) + Hibernate 3.6 (mapeo hbm.xml) + H2 en memoria. CRUD completo de Empleado: listar, buscar, mostrar, añadir, editar y borrar.

Capas de la aplicación

CapaClase / FicheroResponsabilidad
Vistalista.jsp, detalle.jsp, formulario.jspRender HTML; usa JSTL + tags Struts
ControladorEmpleadoCrudDispatchActionRecibe la petición, coordina modelo y vista
Modelo — FormEmpleadoFormTransporta datos del formulario; valida
Modelo — EntidadEmpleadoPOJO mapeado a tabla empleados
Modelo — DAOEmpleadoDaoAcceso a BD mediante Hibernate Sessions
InfraestructuraHibernateUtil, HibernateListenerSessionFactory singleton; lifecycle de la app
