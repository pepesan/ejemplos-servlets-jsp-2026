31 — MVC CRUD: Agenda de contactos

Extiende el patrón Front Controller de 30-mvc con CRUD completo: editar, eliminar, filtrar por categoría, y una vista JSP reutilizable para crear y editar.

Novedades respecto a 30-mvc

ConceptoDescripción
EditarFormulario pre-relleno con datos existentes
EliminarPOST puro con redirect (sin GET que borre datos)
FiltrarLista filtrable por categoría vía ?categoria=
JSP reutilizableformulario.jsp detecta si está en modo crear o editar
Validador extraídoContactoValidator compartido por Crear y Editar

Rutas disponibles

  GET  /app/contactos                    → lista (filtro ?categoria=AMIGO|TRABAJO|FAMILIA)
  GET  /app/contactos/ver?id=X           → detalle
  GET  /app/contactos/nuevo              → formulario vacío (crear)
  POST /app/contactos/nuevo              → guardar nuevo contacto
  GET  /app/contactos/editar?id=X        → formulario pre-relleno (editar)
  POST /app/contactos/editar?id=X        → actualizar contacto
  POST /app/contactos/eliminar (id=X)    → eliminar contacto
