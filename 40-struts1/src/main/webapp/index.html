40 — Struts 1.x: Formulario de saludo

⚠️ Requiere JDK 8 en ejecución. Struts 1.3.x no es compatible con Java 9+.
Primer ejemplo de Struts 1.x. Un formulario con un campo nombre que, al enviarse, muestra un saludo personalizado.

Los tres pilares de Struts 1.x

ElementoClaseFunción
ActionFormSaludoFormContiene los datos del formulario; Struts lo puebla automáticamente desde los parámetros del request
ActionSaludoActionLógica de negocio: valida, construye el saludo y decide a qué vista ir
struts-config.xml—Registra el form-bean, mapea la URL /saludo.do a la Action y declara los forwards

Flujo de la petición

  Navegador
     │  GET /saludo.do (primer acceso)
     ▼
  ActionServlet  (mapeo *.do en web.xml)
     │  Lee struts-config.xml → /saludo → SaludoAction
     │  Crea SaludoForm, puebla campos desde request params
     ▼
  SaludoAction.execute()
     │  nombre está vacío → forward "entrada"
     ▼
  /WEB-INF/vistas/entrada.jsp   (muestra el formulario)

  ─────────────────────────────────────────────

  Navegador
     │  POST /saludo.do  nombre=Ana
     ▼
  ActionServlet  →  SaludoAction.execute()
     │  nombre="Ana" → validación OK
     │  request.setAttribute("saludo", "¡Hola, Ana!")
     │  forward "exito"
     ▼
  /WEB-INF/vistas/resultado.jsp
